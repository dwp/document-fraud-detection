What is a document check?

We check each document against a set of criteria to detect possible document tampering. These checks suggest further areas to examine within your own review.

These checks do not:

  • understand what the document is or what the information means
  • assume relationships between other documents
  • indicate conclusively that fraud is present

If you determine further investigation is needed by a decision maker or investigations team, you may include the information provided in these checks to support your review.

Hidden checks found
Document check Description Recommendations
Damage history
Detected 		This document has been damaged and repaired in the past, which may be a sign that changes were made.
  1. If possible, first contact the claimant to ask for an alternate document.
  2. Cross-reference the other information available to check consistency between this document and claimant information.
  3. Review naming conventions to check for odd or inconsistent file names.
  4. Right-click the document to view document properties or document information, then inspect any file version history or modified timestamps.
  5. If it is available, check if there is a digital IP address, digital signature or any information on where or how the repair occurred.
  6. If you have an IP address, search online to verify whether the document originated from a plausible location (for example, from the United Kingdom).
