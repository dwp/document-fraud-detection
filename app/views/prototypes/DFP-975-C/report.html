How to confirm the checks

Text alignment, size or spacing

Compare how letters, numbers and words line and size up to each other. Anything that seems out of place may be a sign that something was replaced or removed.

Unexpected characters

Select all content and copy it into a text editor like Microsoft Word. If there is a mix of unusual characters and recognisable wording, it may be a sign that changes have been made.

Untrusted branding

Zoom in to search for a second logo or brand that should not be there. For example, a utility bill should not have branding for a freelance or template website.

Security code

This check is not suspicious on its own. Digital signatures are often added to authentic documents. Compare this with other document checks for any signs of editing.
