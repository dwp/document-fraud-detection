What is a document check?
We check each document against a set of criteria to detect possible document tampering. These checks suggest further areas to examine within your own review.

These checks do not:

  • understand what the document is or what the information means
  • assume relationships between other documents
  • indicate conclusively that fraud is present

If you determine further investigation is needed by a decision maker or investigations team, you may include the information provided in these checks to support your review.

Visible checks (9)

You will need to visually inspect the document for signs of tampering by opening and viewing it.

Visible anomalies found
Document check Description Recommendations
Concealed text
Detected 		Text that has been made the same colour as the document background.
  1. Open the document in a document viewer like Adobe Acrobat Reader.
  2. Use the selection tool and the ‘select all’ shortcut (Ctrl + A) to highlight all text and objects in the document to uncover the hidden text.
  3. Zoom in to review any differences in colour on the document.
Edited information
Detected 		Differences in document quality suggest something has been added, edited or removed.
  1. Zoom in around areas that appear blurred, jagged or off-colour. Authentic documents usually have consistent quality throughout (for example, all words or logos being crisp and clear).
  2. Check for differences in the colour or brightness of the document background as this can indicate where new content has been pasted over old content.
  3. Check for signs of manipulation which include inconsistent shadows, mismatched font alignment, and offset text.
  4. Check margins, spacing and paragraph alignment in the document. Inconsistencies may suggest copy and pasted content.
  5. Compare against similar sections, such as headers and tables, to spot inconsistencies.
Hidden objects
Detected 		Original content is hidden by objects that have been placed on top of it.
  1. Open the document in a document viewer like Adobe Acrobat Reader.
  2. Use the selection tool and the ‘select all’ shortcut (Ctrl + A) to highlight all text and objects in the document to uncover the hidden content.
  3. Zoom in to review any differences on the document.
Text alignment
Detected 		The margins between text within the document are inconsistent or unusual.
  1. Check for inconsistencies within headers, addresses, signatures and totals, paying particular attention to the first page. These are frequently altered areas.
  2. Compare left and right text margins. Uneven alignment or misaligned totals may suggest manual editing or copying and pasting from other sources.
  3. Check for consistent text alignment in words and numbers. Changes may indicate that certain words were replaced.
  4. Compare names, dates and amounts for consistency.
  5. Check if the file was emailed or printed. If any of the details are not visible or clear, request a version with full details to check for authenticity.
  6. Compare the document to any verified version you may have available for the same sender or institution.
Text size
Detected 		The sizing of letters or numbers within the document are inconsistent or unusual.
  1. Check for any font size inconsistencies within headers, addresses, signatures and totals, paying particular attention to the first page. These are frequently altered areas.
  2. Check for consistent font sizes between words, letters, and numbers. Changes in letter size may indicate that certain words were replaced.
  3. Compare the document to any verified version you may have available for the same sender or institution.
Text spacing
Detected 		The spaces between text within the document are inconsistent or unusual.
  1. Check for inconsistencies within headers, addresses, signatures and totals, paying particular attention to the first page. These are frequently altered areas.
  2. Check for consistent spacing between words, letters, and numbers. Changes in letter thickness may indicate that certain words were replaced.
  3. Compare the document to any verified version you may have available for the same sender or institution.
Passport standard
Detected 		The Machine Readable Zone (MRZ) of the passport does not exactly match the global standard.
  1. Send the document to the Document Examination Team for further examination.
Unexpected characters
Detected 		Unusual characters or symbols appear in the document.
  1. Copy and paste the text of the document into Microsoft Word and consider:
    * if the text is misaligned or unreadable, the document might be genuine because of how characters have been ‘encoded’
    * if the text pastes as cleanly or perfectly aligned, the document may have been edited or recreated
  2. If possible, cross-check against a previously confirmed document from the same sender or institution to compare if they have similar encoding.
  3. If the document does not appear genuine, check for other signs of manipulation (for example, hidden objects).
Untrusted watermark
Detected 		The document contains an identifying watermark that suggests it was generated from an untrusted source.
  1. Examine the document edges, background or corners for faint text or branding (watermark) that may indicate a document was not created directly by the expected sender.
  2. Use the select all shortcut (Ctrl + A) to highlight objects, annotations or transparent overlays that contain watermark text.
  3. If the watermark refers to a known fake-document site (such as template generators) escalate immediately and keep a reference list of these untrusted sources.
  4. Compare the other document checks for signs of data manipulation (for example, document origin or data changes) before making a judgement.
  5. If uncertain, compare against a verified version of the same document type.

Digital signature
Detected 		A digital signature has been found in the document’s metadata.
  1. This document check is not suspicious alone and should be cross-referenced with other document checks and claimant information.
  2. Consider if it is typical for this type of document to have a digital signature. A digital signature is usually added to the metadata of documents to verify their authenticity. For example, bank statements, tenancy agreements or formal contracts.
Document origin
Detected 		The document was created by untrusted software.
  1. Right-click the document to view document properties or document information, then inspect the creator or producer field.
  2. Cross-reference examples of valid and suspicious combinations of creator and document type. For example, a bank statement created in Microsoft Word should be treated as suspicious.
  3. Check for messaging, social media or screenshot creation tags that suggest that the document came from another source or was captured from a screenshot.
  4. If possible, ask the claimant for alternate evidence if the document is a screenshot or saved through messaging or social media.
